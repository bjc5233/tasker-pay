# tasker-pay



## 说明
左下角快捷弹出菜单
* 安装tasker
* 导入tasker项目
* [pay]下配置根路径[root_path]变量为自己的路径，并在路径中放入[pay]文件夹中所有图片
* 在桌面创建该任务的快捷方式
* 点击快捷方式弹出界面，再次点击快捷方式退出；也可以使用tsf桌面手势配置[手指上滑执行pay任务]
* 点击菜单执行对应的支付方式
